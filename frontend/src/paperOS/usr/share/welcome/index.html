welcome to paperOS V.0.2!

paperOS is built ENTIRELY with javascript and javascript alone. Develop apps with the intuitive POK Kernel, have the flexibility of a cat at your fingertips; and really make your operating system yours.

What is new?

1. Upgraded file system!

We are now using a cache system, so you can store even more files then you could previously. Along with that; developers have an easier time accessing whatever files for their programs they would like! No longer do you have to go through window.pok.fileSystem.readFile to read files! Alongside with that; we have plans for allowing paperOS to read the files on your device; separated out into drives.
btw; firefox users; if you in the past have experienced any lag when using 'ls', don't worry! that's all gone now with the swith to the cache system. Enjoy browsing your files with speed and efficiency!

2. Added a Network Driver

We also have plans for a web browser and other internet related tasks; this is the first major step to achiving such a thing.
